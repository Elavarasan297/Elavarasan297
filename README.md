📌 Featured Projects

📊 Titanic Survival Prediction

End-to-end ML classification pipeline on 891 passenger records — data cleaning, feature engineering (FamilySize, IsAlone, group-wise age imputation), and model comparison between Logistic Regression and Random Forest. Achieved 80.4% test accuracy, with sex, fare, age, and passenger class identified as the strongest predictors via feature importance analysis.

Tech Stack: Python Pandas scikit-learn Matplotlib Seaborn

🔗 Repository: github.com/Elavarasan297/titanic-survival-prediction
